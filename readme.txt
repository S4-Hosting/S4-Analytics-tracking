== Description ==

This unofficial, simple plugin helps you add a tracking code for Matomo to your WordPress website, for an easier integration.

For more information about Matomo, visit [matomo.org](https://matomo.org/).

== Installation ==

This section describes how to install the plugin and get it working.

1. Install and activate the plugin as you normally would. (You can find more details and instructions [here](https://wordpress.org/support/article/managing-plugins/).)
2. Visit the "Tracking code for Matomo" page in the "Settings" section and adjust the parameters. Don't forget to click "Save Changes" when you're done.
